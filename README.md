# FlowIndexer

Flow Indexer is an infrastructure layer that aggregates, indexes, and visualizes data from funding streams and trust networks. It tracks transactions on Stellar and streaming flows, transforming raw blockchain data into meaningful insights.

The platform provides APIs, dashboards, and graph visualizations that show who is funding whom, how funds move through networks, and which entities have the greatest impact. By making funding flows transparent and analyzable, Flow Indexer improves decision-making for developers, funders, and communities.

As a developer-focused tool, it also enables other applications to build on top of its data layer, making it a critical piece of infrastructure for the broader ecosystem.
